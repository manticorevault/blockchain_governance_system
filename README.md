> MGMTLinks is a blockchain-based management system. Manage your project's requests, your contributors' vote counts and more! Built in a decentralized way, on top of the Ethereum blockchain, this application allows teams and organizations to better manage their project's governance.

## Cloning
```sh
git clone https://github.com/manticorevault/blockchain_governance_system/
```

## Installing

```sh
npm install
```

## Running tests

```sh
npm run test
```

## Compiling the Smart Contract

```sh
cd ethereum
```

```sh
node compile.js
```

(If you get any errors, check for solc version on package.json - If it's not 0.4.17, then uninstall it (npm uninstall solc) and install it again in the right version - npm install solc@0.4.17)

```sh
node deploy.js
```
